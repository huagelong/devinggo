devinggo

🚀一款基于 Vue 和 Goframe v2 的全栈开发平台，集成 JWT 鉴权、动态路由、消息队列等功能，提供丰富场景模板，助您快速构建企业级应用，专注业务开发。

开始使用 →

功能丰富

内置了完整的权限体系，包括：菜单、用户、角色、部门等权限控制。也包括了日常的系统监控、字典代码。还有代码生成器等功能。

流行的技术栈

以 Vue 和 Goframe v2 为核心技术栈，结合流行的前端框架和后端框架，提供丰富的功能和高效的开发体验。

高颜值后台系统

前端基于mineadmin v2版本，提供高颜值后台系统。
