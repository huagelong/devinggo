目录结构

主要目录结构:


├── docs                    # 文档目录
├── api                     # API定义
├── hack                    # hack工具和配置
├── internal                # 内部
├── manifest
│   └── config             # 配置文件目录
├── modules                # 业务模块目录
│   ├── _                   # 模块嵌入入口
│   ├── system            # 系统模块
│   └── api                # API模块
├── resource              # 资源文件
│   ├── generate         # 代码生成模板
│   └── public          # 静态资源
└── web                  # 前端代码
    └── admin          # 系统前端
